# 🖥️ KTU S6 Python GUI Programs

**Kerala Technological University | Semester 6 | Python Lab**  
GUI-based Python programs built using **Tkinter** and related libraries.

---

## 📁 Programs

| No. | Program | Description |
|-----|---------|-------------|
| 1 | `calculator.py` | Basic arithmetic calculator with GUI |
| 2 | `file_manger.py` | Manages all sort of basic operations on files (Add/View/Delete) |
| 3 | `student_record.py` | Student record management (Add/View/Delete) |
| 4 | `login_form.py` | Login and registration form with validation |
| 5 | `to_do_list.py` | To-Do list app with add/remove tasks |
| 6 | `text_editor.py` | Simple text editor with file open/save |
| 7 | `paint_app.py` | Drawing/paint application using Canvas |
| 8 | `quiz_app.py` | Multiple choice quiz with score tracking |
| 9 | `currency_converter.py` | Currency converter with dropdown selection |
| 10 | `digital_clock.py` | Real-time digital clock display |
| 11 | `bmi_calculator.py` | BMI calculator with result display |

---

## ⚙️ Requirements

- Python 3.x
- Tkinter (comes built-in with Python)

To verify Tkinter is available:
```bash
python3 -m tkinter
```

---

## 🚀 How to Run

```bash
# Run any program directly
python3 calculator.py
```

> No additional installations needed — Tkinter is part of the Python standard library.

---

## 🧰 Tkinter Widgets Used

- `Label`, `Entry`, `Button` — basic UI elements
- `Frame`, `LabelFrame` — layout containers
- `Canvas` — for drawing (paint app)
- `Listbox`, `Scrollbar` — for record/list views
- `Menu`, `Menubar` — for text editor menus
- `OptionMenu`, `Combobox` — for dropdowns
- `messagebox` — for popups and alerts
